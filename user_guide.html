Child Tasker App's User Guide

Last updated Nov 16, 2020

Registering User

When you first install Child Tasker app on your device, the app will take you to register page where you can register as a parent

  1. Enter your name/username/nickname
  2. Enter your pin/password
  3. Re-enter your pin/password
  4. Click Register

Adding a Child

To create a new child account:

  1. From main screen i.e., "Parent Dashboard", click on "Manage Child Details" which will take you to "Manage Child Dashboard"
  2. From here, click on "Add New Child".
  3. Enter your child name/nickname
  4. Click on "Generate New Pin"
  5. Please, remember this 4 digit pin and pass it to your child
    Note: Using the name and the pin number, your child can access their account.
  6. Click on "Save" to create an account for the child

Adding a Category

To create a new category which is collection of tasks:

  1. From main screen i.e., "Parent Dashboard", click on "Manage Categories & Tasks"
  2. This will take you to "Categories and Tasks" screen
  3. From here, click on "Add New Category"
  4. Enter name you would like for the category
  5. Select an icon for the category
  6. Select on background color for the icon
  7. Click on "Save" to create a new category

Adding a Task

To add new task:

  1. From main screen i.e., "Parent Dashboard", click on "Manage Categories & Tasks"
  2. This will take you to "Categories and Tasks" screen
  3. From here, click on "Add New Task"
  4. Enter name for the task
  5. Enter points for the task
  6. Select an icon for the task
  7. Select background color for the icon
  8. Assign the task to the category it belongs to
  9. Click on "Save" to create a new task

Assigning Tasks and Reward to a Child

To assign tasks and reward to a child:

  1. From main screen i.e., "Parent Dashboard", click on "Create Tasks For Child"
  2. This will take you to "Create Tasks For Child" screen
  3. Select a child for whom you would like to assign rewards and tasks
  4. Select reward you would like to assign to that child
  5. Select categories
  6. Select task that you would like to assign
  7. Click on "Add Another Task"
  8. You have to repeat selecting categories and tasks and adding another tasks until tasks' total point is equal or greater than reward point
  9. When tasks' total is greater or equal than reward total "Save" button will appear
  10. Click on "Save" to assign list of tasks and reward to a child

Viewing Child's Progress

To viee child's progress:

  1. From main screen i.e., "Parent Dashboard", click on "Manage Rewards"
  2. This will take you to "Manage Reward Dashboard" screen
  3. Click on "Track Reward"
  4. Select the child whose progress you want to view
  5. Select the reward you are interested in
  6. Then you will be presented with a pie-chart showing completed tasks and incomplete tasks for that reward

Completing a Task (By child)

When a child completes a task:

  1. As a child, you should login using your name/username and pin number assigned to you
  2. Once login is successful, you will be taken to "Reward Dashboard" screen
  3. Then you should select the reward for which the task belongs
  4. Or if you just want to view what tasks you need to complete to get that reward, you can click the reward
  5. Now you can view all tasks involved for that reward
  6. If you want to complete a task, click on the task you have completed
  7. change the toggle "Completed" button from "No" to "Yes"
  8. Click on "Save"
  9. You will be taken to the "Progress Screen" where you can view your progress for that reward
